# 🕵️ The Side Hustle Scout

A lightweight, high-performance web application designed to help teenagers and young adults find a realistic side hustle based on their existing equipment, skills, and time availability.

## ✨ Features
- **Zero-AI Logic:** Runs entirely on a custom decision-tree algorithm for instant results and privacy.
- **Mobile-First Design:** Optimized for social media sharing with a Gen-Z "Dark Mode" aesthetic.
- **Gamified Experience:** Includes progress tracking, haptic-style feedback, and a confetti celebration upon completion.
